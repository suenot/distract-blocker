Distract Blocker — небольшое и сосредоточенное Chrome-расширение, которое блокирует отвлекающие сайты. Без рекламы, без телеметрии, без аккаунта.

Весь интерфейс живёт в боковой панели, которая выезжает при клике на иконку. Всё в одном месте: тоггл одним касанием, список сайтов, расписание, выбор языка, дружелюбный маскот-страж. Никакого popup, исчезающего при первом же движении мыши. Никаких настроек, спрятанных за пять кликов.


★ ОСНОВНЫЕ ВОЗМОЖНОСТИ

• Боковая панель — тоггл, редактирование списка, расписание. Не закрывается, пока вы переключаете вкладки.

• Два режима редактирования списка — добавлять сайты по одному (List) или вставить весь список текстом (Text). Поддомены блокируются автоматически.

• Расписание — три режима:
  – Always: блокирует, пока включён тоггл.
  – Block during: блокирует только в выбранные интервалы (дни недели + время). Удобно для рабочих часов.
  – Allow during: блокирует всегда, кроме выбранных интервалов. Удобно для контролируемого «окна на отдых».

• Свой горячий шорткат — задаётся в chrome://extensions/shortcuts. Мгновенно переключает блокировку, не открывая никакой UI — на тот самый случай, когда вам действительно надо быстро глянуть.

• 21 язык — English, Русский, 中文, 日本語, 한국어, Español, Português, हिन्दी, العربية (с поддержкой RTL), Français, Deutsch, Italiano, Nederlands, Türkçe, Қазақша, Oʻzbek, Кыргызча, Bahasa Indonesia, Tiếng Việt, ไทย, Bahasa Melayu. Определяется по браузеру, переключается в шапке панели.

• Дружелюбная страница «не сейчас» — вместо системной ошибки Chrome заблокированные сайты редиректятся на маленькую страницу внутри расширения, со сонным маскотом и кнопкой «назад».


★ КАК ЭТО РАБОТАЕТ

Блокировка работает через нативный API Chrome declarativeNetRequest. Никаких content-script-инъекций, никакого удалённого кода, никаких сетевых запросов наружу. Всё локально: ваш список, ваше расписание, ваш язык. Расширение не видит содержимое страниц, которые вы посещаете.


★ ДЕФОЛТЫ

Расширение поставляется с разумным стартовым списком (YouTube, Facebook, Instagram, Twitter / X, TikTok, Reddit, VK, OK), который можно полностью заменить. Тоггл по умолчанию включён.
